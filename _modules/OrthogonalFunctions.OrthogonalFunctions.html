OrthogonalFunctions.OrthogonalFunctions.OrthogonalizationUtilities.CheckMutualOrthonormality(phi_orthonormalized_list, Interval)[source]¶

Checks the inner orthonormality of each of two fuctions froma list of symbolic functions.

It returns an array with elements -1, 0, or 1.

  • -1: two functions are not orthogonal.

  • 0: two functions are orthogonal.

  • +1: function is orthonormal.

Parameters

  • phi_orthonormalized_list (list) – The list of sympy functions that are orthonormalized.

  • Interval (list) – The interval of the domain of the functions in the form [Start,End].

Returns

The mutual orthogonality matrix.

Return type

ndarray

OrthogonalFunctions.OrthogonalFunctions.OrthogonalizationUtilities.PrintCoefficientsOfFunctions(phi_orthonormalized_list, StartFunctionIndex)[source]¶

Prints the coefficients of orthonormalized functions as

\[\phi_j(t) = \alpha_j * \sum_{i=1}^n a_{ij} t^{\frac{1}{i}}\]

where \(\alpha_j = \sqrt{\frac{2}{j}}\), and \(a_{ij}\) are integers.

Parameters

  • phi_orthonormalized_list (list) – The list of sympy functions that are orthonormalized.

  • StartFunctionIndex (int) – The start index of the functions.
